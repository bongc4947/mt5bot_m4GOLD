## What is NOT in this number

The Tester models broker spread and order execution, but it does NOT
fully model:

- **Real-time slippage** on fast moves (AvaTrade demo executes at the
  modeled price every time)
- **Requotes / partial fills** (production broker can reject orders)
- **Swap costs on held positions** — MetaTrend holds for hours so swap
  matters; Tester does charge swap but its rate may differ from live
- **News-event gaps** in real time vs Tester's reconstructed bars

Expect ~0.05-0.10 additional PF degradation live. That puts the live
expectation at roughly **PF 1.15-1.20**, which is still profitable and
matches the original ship-gate threshold (`gate.min_pf = 1.15`).

## Behavioural sanity check

- **608 round-trips / 6 months ≈ 3.4 trades per trading day.** That's
  more active than I'd have guessed for a 50/200 EMA-cross trend rule.
  The meta-gate flicks above/below 0.55 around EMA flips, and
  `InpExitOnFlip=true` closes on every primary direction change. If we
  want to reduce churn, the obvious knob is to require a sustained
  P(act) > 0.55 over N consecutive bars before entry, OR to bump the
  threshold to 0.60.
- **Win rate 44.7% with R/R 1.54.** Healthy trend-following profile.
  Most trades give back a little, a smaller number of trades capture
  the trend body. This is the EXPECTED shape for this strategy class.
- **Worst trade -$217.21.** That's 21.7x the avg loss and roughly the
  3-ATR initial stop on a $4000+ GOLD price. Normal.
- **Drawdown 3.4%.** Well within the leverage envelope of 0.01 lots on
  a $10k deposit. Could be levered up but the next field-test should be
  on demo for a few weeks before doing that.

## Reproduce

```
# in repo root
$mt5 = "C:\Program Files\MetaTrader 5\terminal64.exe"
$ini = "tester_reports\tester.ini"   # generated by the workspace
Start-Process $mt5 -ArgumentList "/config:`"$ini`"" -Wait
python python/parse_tester_log.py
```

The HTML report flag (`Report=...`) in tester.ini did NOT produce a
file with the MT5 build 5836 we ran; the agent log was parsed instead.
That's a known finickiness with absolute paths in tester.ini. The
agent log is the authoritative source either way — its trade-by-trade
stream is what MT5 itself would aggregate.
